진정 화장품 포지셔닝맵

빅데이터 컴퓨팅 5조

인터랙티브하게 줌인, 줌아웃, 이미지위에 마우스를 올리면 뜨는 팝업 기능을 이용해서 상세한 정보를 알 수 있습니다. 아래는 두가지 시각화 페이지입니다.

1.상품 포지셔닝 맵 페이지

제품 선정 기준 : 올리브영 랭킹 100위 안에 있는 제품군 중, 리뷰 데이터를 크롤링해 진정에 도움이 된다라고 리뷰한 퍼센트가 30%이상인 제품을 진정제품으로 분류하여 포지셔닝 하였습니다.

사용한 데이터
1. 사진: 해당하는 상품의 썸네일
2. 사진의 사이즈 : 지난 6개월간 광고를 제외한 해당 상품에 관한 유투브 콘텐츠의 댓글수 합산
3. 퀄리티 : 랭킹100위내 해당 브랜드 상품에 대한 올리브영 리뷰수 + 지난 6개월간 광고를 제외한 해당 상품에 관한 유투브 콘텐츠의 총 댓글수 합산
4. 가격 : 해당 상품들의 평균 가격

2.제품의 브랜드 포지셔닝 맵 페이지

브랜드 선정 기준 : 분류한 진정제품을 순위로 20등까지 나열하였고, 해당 제품들을 보유하고 있는 브랜드는 결과적으로 11개가 나왔습니다.

사용한 데이터
1. 원의 사이즈 : 브랜드 6개월 매출액
2. 원의 색깔 : 올리브영 랭킹(보라색일수록 순위가 높음, 즉 1에가까움)
3. 평균리뷰개수 : 올리브영 랭킹100위내 해당 브랜드 상품에 대한 리뷰수의 평균
4. 평균좋아요/조회수 : 올리브영 랭킹100위내 해당 브랜드 상품에 대한 유튜브 콘텐츠 평균 좋아요/조회수
